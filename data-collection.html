We scraped and filtered tweets as our data.

For the data collection section of this project, the team gathered tweets containing misinformation and disinformation about the topic, COVID-19 vaccines contain microchip, magnetic objects, and other substances. The dataset for this initiative can be accessed here. To discuss how the data collection process went through, we answer the following questions:

How did you collect your data?

There are two different methods employed to gather tweets from Twitter. The first method uses the advanced search feature of Twitter where tweets are searched based on keywords and post date. The keywords used include "vaccine", "kutsura", "laman", "chip", "bakuna", "barya", "microchip", "track", "coin", etc.

The collector then manually collates all necessary features of chosen tweets into the spreadsheet. The second method uses scraping script implemented on python. Similarly, this tool gathers tweets with matching keywords and date while creating a csv file contaning the tweets and their features. The collectors filter the desired tweets manually before transferring it to the sheet.

When did you collect you data?

Before collecting the data, some sample tweets were obtained during the last week of February 2023 and the first week of March 2023, which is necessary to gauge if the topic is feasible. After having the topic proposal approved by our professor, we collected most of the data from 28 March 2023 to 18 April 2023. After the data collection, the tweets were subjected to review by different reviewers to check if the data is correct, complete, clean, and unique.

What's your selection criteria for collecting data?

As mentioned earlier, the tools used by the team filters the tweets based on keywords and timeframe. Since the pandemic started on March 2020, the timeframe considered started on that date. These tools serves as the preliminary filter to ensure that the tweets are in line with the topic. Then, the team manually checks all these filtered tweets whether they contain disinfo or misinfo, and are related to the topic. Before placing the filtered tweets into the spreadsheet, the collectors check if the tweets are fact-checkable, relevant, and verifiable using reliable sources.

How many samples did you collect?

The team collected a total of 151 samples where 73 tweets are labelled Filipino and 78 tweets are labelled as Foreign. To clarify, Filipino tweets are contained in threads that use Filipino language, while Foreign tweets are contained in threads that did not use Filipino language.

What are the data features? Describe each one.

The data features of the Clean Dataset are as follows:

  • ID - has format GG-ID, where GG corresponds to the group collector and ID corresponds to the assigned identifier

  • Timestamp - date and time this tweet was collected in Philippine Standard Time of format DD/MM/YY HH:MM:SS

  • Tweet URL - complete URL of the tweet. When clicked, it automatically redirects to the original tweet

  • Group - the assigned group number of the team in which the collector of this tweet is part of

  • Keywords - the keywords fed onto the advanced search and scraping tool used when searching for this tweet

  • Account Handle - the account handle of the user that tweeted this tweet

  • Account Name - the account name of the user that tweeted this tweet

  • Account Type - the account type of the user which can either be identified, anonymous, or media

  • Tweet - the tweet itself in raw text format with an optional alt-text if the tweet contains media

  • Tweet Translated - the English translation of the raw text format of this tweet

  • Tweet Type - the tweet's label; can be one or more of the following: text, image, video, URL, reply

  • Date Posted - ate and time this tweet was posted in Philippine Standard Time of format DD/MM/YY HH:MM

  • Content Type - the content's label which can either be emotional, rational, or transactional

  • Reasoning - a brief explanation as to why the tweet is misinformation or disinformation

  • Thread/Tweet Language - the thread's lable which can either be Filipino or Foreign depending on the language
